## 📝 章节格式规范

**重要：** 所有章节必须遵循统一格式！

### 标准格式

```markdown
# 每天偷一小时

## 第 X 章 章节标题

（空一行）

正文段落（每段 100-300 字，段间空一行）

（空一行）

"对话内容。"人物说。

（空一行）

---

**（第 X 章 完）**

**字数统计：** 约 XXXX 字

**下一章预告：** 简短预告
```

### 格式要点

- ✅ 段落之间**空一行**
- ✅ 对话**独立成段**
- ✅ 场景转换用 `---` 分隔
- ✅ 章节结尾格式统一
- ❌ **不要频繁换行**（每句都换行是错误的）
- ❌ **不要段落太长**（超过 500 字需要分段）

**详细规范：** 查看 `章节格式规范.md`

## 🎯 创作流程

定时任务触发时，AI 会：

1. **读取进度** - 查看 `写作进度.json` 确定要写第几章
2. **查阅大纲** - 从 `MEMORY.md` 或 `完整大纲.md` 获取章节信息
3. **创作章节** - 根据大纲要求写 5000 字+
4. **保存文件** - 保存为 `第 X 章_标题.md`
5. **更新进度** - 更新 `写作进度.json`
6. **报告完成** - 向舰长报告本章完成情况和下一章预告
