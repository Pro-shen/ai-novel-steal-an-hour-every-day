## 🌍 世界观设定

### 隙间（The Gap）

| 规则 | 说明 |
|---|---|
| **时间** | 每天 00:00-01:00，现实世界暂停，隙间开启 |
| **进入者** | 约 0.1% 的人能进入，称为"隙间人" |
| **可见性** | 隙间人互相可见，普通人看不见隙间人 |
| **物理法则** | 部分物理法则失效（如电子信号不稳定） |
| **物品携带** | 隙间内拿的东西，可以带到现实 |
| **时间悖论** | 隙间内杀人，现实中会死亡，但死因无法解释 |

### 隙间人分类

| 类型 | 说明 |
|---|---|
| **自然觉醒** | 天生能进入，如主角林默 |
| **诱导觉醒** | 被其他隙间人唤醒 |
| **组织成员** | "时盗"组织，利用隙间犯罪 |
| **猎手** | 追查隙间犯罪的秘密组织 |
