# 《每天偷一小时》章节格式规范

## 📝 格式标准

### 1. 标题格式

```markdown
# 每天偷一小时

## 第 X 章 章节标题
```

### 2. 正文格式

- **段落间距**：每段之间空一行
- **段落长度**：每段 100-300 字，不要频繁换行
- **对话格式**：独立成段，不要和其他描述混在一起
- **场景转换**：用 `---` 分隔线标记

### 3. 章节结尾

```markdown
---

**（第 X 章 完）**

**字数统计：** 约 XXXX 字

**下一章预告：** 简短预告内容
```

## ✅ 正确示例

```markdown
# 每天偷一小时

## 第一章 不可能的犯罪

凌晨两点，林默站在案发现场，盯着那具尸体已经十分钟了。

死者是个中年男人，仰面倒在书房中央，胸口插着一把水果刀。血已经凝固，在深色地毯上晕开一片更深的痕迹。门窗完好，监控显示昨晚没有任何人进出这栋别墅。

"林队，技术科的结果出来了。"助手小张走过来，声音有些发紧，"死亡时间……无法确定。"

林默转过头："什么意思？"

---

**（第一章 完）**

**字数统计：** 约 5000 字
```

## ❌ 错误示例

```markdown
# 每天偷一小时

## 第一章 不可能的犯罪

凌晨两点。
林默站在案发现场。
盯着尸体。
十分钟了。  ← 频繁换行，错误

死者是个中年男人。仰面倒地。胸口插刀。 ← 句子太碎，错误

---

**（第一章完）**  ← 格式不统一，错误
```

## 📋 格式检查清单

每章完成后检查：

- [ ] 标题格式正确（# 每天偷一小时 / ## 第 X 章）
- [ ] 段落之间空一行
- [ ] 对话独立成段
- [ ] 场景转换用 --- 分隔
- [ ] 章节结尾格式统一
- [ ] 字数统计
- [ ] 下一章预告

- **每章目标**：5000 字以上
