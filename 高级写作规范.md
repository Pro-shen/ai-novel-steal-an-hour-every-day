# 《每天偷一小时》高级写作规范

## 🎨 核心写作原则

### 1. 展示，不要告诉 (Show, Don't Tell)

**❌ 错误：**
```
林默很紧张。
```

**✅ 正确：**
```
林默的手指在桌面上轻轻敲击，一下，两下，三下。他停下，发现自己已经在不知不觉中敲出了心跳的节奏。
```

### 2. 对话要有潜台词

**❌ 错误：**
```
"你害怕吗？"小张问。
"我不害怕。"林默说。
```

**✅ 正确：**
```
"林队，手怎么抖了？"小张盯着他按在桌沿的手指。

林默把手收回来，插进兜里。"冷的。"

办公室里，暖气开得很足。
```

### 3. 感官细节要丰富

**五感运用：**

| 感官 | 示例 |
|---|---|
| 视觉 | 颜色、光影、形状、运动 |
| 听觉 | 声音、音调、节奏、沉默 |
| 嗅觉 | 气味、味道、空气质感 |
| 触觉 | 温度、质地、压力、疼痛 |
| 直觉 | 预感、不安、熟悉感 |

**✅ 正确示例：**
```
空气里有股味道。不是血，不是尸体腐烂，而是某种更古老的东西——像图书馆里泛黄的书页，像地下室里积灰的铁盒，像时间本身的气味。
```

### 4. 节奏要有变化

**快节奏：** 动作戏、追逐、紧张时刻
- 短句
- 少描写
- 多动词

**慢节奏：** 思考、回忆、情感时刻
- 长句
- 多描写
- 留白

**示例：**
```
快：他跑。楼梯。转角。撞到人。道歉。继续跑。

慢：林默站在窗前，看着外面的雨。雨滴在玻璃上留下痕迹，一道，又一道，像是有人用手指在上面写字。写的是什么？他看不清。
```

### 5. 人物要有层次

**表面层：** 职业、身份、外表
**中间层：** 性格、习惯、喜好
**核心层：** 恐惧、欲望、秘密

**每章至少展现一个层次：**
```
表面：林默是刑警队长，冷静专业
中间：他有失眠症，靠数心跳入睡
核心：他害怕失去，因为已经失去过妹妹
```

### 6. 悬念要分层

**即时悬念：** 本章内解决
- 死者手里有什么？

**中期悬念：** 10-20 章解决
- 妹妹在哪里？

**长期悬念：** 全书结束解决
- 隙间到底是什么？

### 7. 细节要有意义

**❌ 无用细节：**
```
他穿了一件蓝色的衬衫。（然后再也没有提到）
```

**✅ 有意义细节：**
```
他穿了一件蓝色的衬衫，袖口磨得发白。妹妹送的，她说蓝色适合他。十年了，他还在穿。
```

### 8. 情感要克制

**❌ 过度煽情：**
```
他悲痛欲绝，眼泪止不住地流，心碎成了碎片。
```

**✅ 克制表达：**
```
他看着照片，看了很久。然后把照片翻过去，扣在桌上。
```

## 📝 每章写作检查清单

### 开篇（前 500 字）
- [ ] 有吸引读者的钩子
- [ ] 建立场景氛围
- [ ] 引入至少一个感官细节

### 中段（3000-4000 字）
- [ ] 对话有潜台词
- [ ] 展现人物层次
- [ ] 节奏有变化
- [ ] 有可品味的细节

### 结尾（后 500 字）
- [ ] 有悬念钩子
- [ ] 情感有余韵
- [ ] 让读者想翻下一页

### 全文
- [ ] 展示多于告诉
- [ ] 五感运用至少三种
- [ ] 有至少一个让人记住的细节
- [ ] 有留白让读者思考

## 🎯 可品读内容标准

**什么是可品读的内容？**

1. **有回味的对话** - 读完想再看一遍
2. **有深意的细节** - 看似普通，后有深意
3. **有共鸣的情感** - 读者能感同身受
4. **有思考的空间** - 不是直给，需要品味

**示例：**

```
❌ 不可品读：
"我要找到妹妹。"林默说。

✅ 可品读：
林默摸出钱包，里面夹着一张照片。十六岁的女孩，笑得很灿烂。

"十年了。"他说，"她应该二十六了。"

照片的边角已经磨圆了。
```

## ✍️ 修改流程

每章写完后：

1. **第一遍** - 检查情节逻辑
2. **第二遍** - 增加感官细节
3. **第三遍** - 优化对话潜台词
4. **第四遍** - 调整节奏变化
5. **第五遍** - 添加可品读细节

**标准：** 至少修改 5 遍才能发布
